# Derlis Aguilera Portfolio

Portfolio profesional para la marca personal de Derlis Aguilera como **Ecommerce Strategy & AI Operations Specialist**.

## Arquitectura

- `app/`: rutas, metadata SEO y estilos globales con App Router de Next.js.
- `components/`: componentes reutilizables de presentación y animación.
- `lib/content.ts`: contenido estructurado para experiencia, proyectos, stack y pilares.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run typecheck
```
