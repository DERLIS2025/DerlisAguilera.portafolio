# Derlis Aguilera Portfolio

Portfolio profesional orientado a resultados para la marca personal de Derlis Aguilera como **Ecommerce Strategy & AI Operations Specialist**. El sitio prioriza impacto, casos de negocio, métricas personales, AI Operations y arquitectura digital para que recruiters y empresas entiendan rápidamente su combinación de negocio, ecommerce, automatización e inteligencia artificial.

## Arquitectura

- `app/`: rutas, metadata SEO y estilos globales con App Router de Next.js.
- `components/`: componentes reutilizables de presentación y animación.
- `lib/content.ts`: contenido estructurado para KPIs, casos de negocio, arquitectura, capturas conceptuales, evolución profesional, experiencia, proyectos, stack y pilares.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run typecheck
```
